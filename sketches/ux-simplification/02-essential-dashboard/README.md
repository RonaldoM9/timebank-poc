## Variante 2 : Essential Dashboard

### Design stance
Un vrai tableau de bord, mais **réduit à l'essentiel** : 3 indicateurs max, pas de gamification intrusive, flux de lecture vertical clair.

### Key choices
- **Welcome card** unique : avatar + localisation + niveau + solde TIME en un bloc
- **3 stat cards** au lieu de 5 : réservations, demandes, pot commun — les 3 métriques qui comptent
- **Suppression** : progression hero (XP), badges, alerte bannière, collectives, footer comics
- **Navigation desktop** : barre de liens dans le header (pas de menu hamburger)
- **2 CTA** bien distincts : Explorer (secondaire) / Proposer (primaire vert)

### Trade-offs
- **Fort** : lisible en 3 secondes, donne envie d'agir, reste un "dashboard" rassurant
- **Faible** : moins de "gratification" immédiate (pas de barre XP, pas de badges)
- **Compromis** : garde une vue d'ensemble sans submerger

### Best for
- Utilisateurs réguliers qui veulent un coup d'œil rapide
- Profils institutionnels / financeurs qui veulent voir le sérieux de la plateforme
