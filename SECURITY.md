# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in TimeHeroes / timebank-poc, please report it privately.

**Do not open a public GitHub issue.**

Contact Ronald directly:
- **Email:** mounien.ronald@gmail.com
- **Response time:** Within 48 hours

## Scope

This project is a proof-of-concept (POC) for TimeHeroes, a time-banking application.
The following are in scope:
- Smart contract logic (if applicable)
- API endpoints and authentication
- User data handling and privacy
- Database security

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅ Yes    |

## Disclosure Policy

We practice responsible disclosure:
1. Reporter submits vulnerability details
2. We acknowledge receipt within 48 hours
3. We work on a fix and notify reporter
4. Once fixed, we coordinate public disclosure

Thank you for helping keep TimeHeroes secure.
